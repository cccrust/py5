# v1.2 規劃：pip client/server 功能

## 目標

建立私有 pip server + client，讓 py5 可以上傳和安裝套件。

## 架構

```
py/
  pip/
    client.py     # pip client (py5 pip install/upload/list)
    server.py     # pip server (HTTP API + 儲存)
    storage/      # 套件儲存目錄
```

## 功能清單

### 1. pip client (client.py)

| 命令 | 說明 |
|------|------|
| `py5 pip install <pkg>` | 從 server 下載並安裝套件 |
| `py5 pip upload <pkg>` | 打包並上傳套件到 server |
| `py5 pip list` | 列出已安裝的套件 |
| `py5 pip search <keyword>` | 搜尋 server 上的套件 |
| `py5 pip remove <pkg>` | 移除已安裝的套件 |

### 2. pip server (server.py)

| API | 方法 | 說明 |
|-----|------|------|
| `/` | GET | 列出所有可用套件 |
| `/<pkg>` | GET | 下載指定套件 |
| `/upload` | POST | 上傳新套件 |
| `/<pkg>/<ver>` | DELETE | 刪除套件 |

### 3. 執行方式

```bash
# 啟動 server
./py5 run py/pip/server.py

# client 操作
./py5 pip upload mypackage
./py5 pip install mypackage
```
